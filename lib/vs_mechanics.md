# Механики Vampire Survivors

## Основные системы:

### 1. Оружие (Weapons)
- Автоматически атакуют врагов
- Имеют разные паттерны атаки
- Можно улучшать до уровня 8
- При достижении макс. уровня + пассив = эволюция

### 2. Пассивные улучшения (Passives)
- Улучшают характеристики
- Можно комбинировать с оружием для эволюции
- Имеют максимальный уровень

### 3. Система опыта
- Опыт падает с врагов
- Автоматически собирается в радиусе
- Магниты увеличивают радиус сбора

### 4. Эволюция оружия
- Оружие макс. уровня + определенный пассив = эволюция
- Эволюция намного сильнее базового оружия

### 5. UI
- Отображение активного оружия и пассивов
- Иконки с уровнями
- Красивые окна выбора улучшений
