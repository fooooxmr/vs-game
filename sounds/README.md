# Звуки для игры

## Как скачать звуки

Звуковые файлы нужно скачать вручную с открытых источников. Рекомендуемые источники:

1. **OpenGameArt.org** - https://opengameart.org/content/rpg-sound-pack
2. **Freesound.org** - https://freesound.org (требуется регистрация)
3. **Kenney.nl** - https://kenney.nl/assets/audio-pack
4. **Pixabay** - https://pixabay.com/sound-effects/

## Необходимые звуки

Скачайте следующие звуки в формате WAV и поместите их в эту папку:

- `attack.wav` - звук атаки
- `enemy_hit.wav` - звук попадания по врагу
- `enemy_death.wav` - звук смерти врага
- `level_up.wav` - звук повышения уровня
- `pickup.wav` - звук подбора предмета
- `chest_open.wav` - звук открытия сундука
- `upgrade_select.wav` - звук выбора улучшения
- `player_hit.wav` - звук получения урона игроком
- `boss_spawn.wav` - звук появления босса
- `elite_attack.wav` - звук атаки элитного врага
- `projectile_shoot.wav` - звук выстрела
- `barrel_explode.wav` - звук взрыва бочки

## Альтернативный способ

Можно использовать скрипт `download_sounds.sh` (если он работает) или скачать готовый набор звуков с одного из указанных источников.
