# Звуковые эффекты

Для работы звуков в игре, поместите следующие файлы в папку `sounds/`:

## Необходимые звуковые файлы:

- `attack.wav` - звук атаки (кнут)
- `enemy_hit.wav` - звук попадания по врагу
- `enemy_death.wav` - звук смерти врага
- `level_up.wav` - звук повышения уровня
- `pickup.wav` - звук сбора опыта/предметов
- `chest_open.wav` - звук открытия сундука
- `upgrade_select.wav` - звук выбора улучшения
- `player_hit.wav` - звук получения урона игроком
- `boss_spawn.wav` - звук спавна босса
- `elite_attack.wav` - звук атаки элитного монстра
- `projectile_shoot.wav` - звук выстрела снаряда
- `barrel_explode.wav` - звук взрыва бочки
- `music.ogg` или `music.wav` - фоновая музыка (опционально)

## Где скачать звуки:

1. **SoundEffect.app** - https://soundeffect.app/ru
2. **ZvukBox** - https://zvukbox.ru/
3. **Pixabay** - https://pixabay.com/ru/sound-effects/search/game/
4. **Free Sounds Library** - https://free-sounds-lib.ru/music/games-effects/

## Формат файлов:

- Поддерживаются форматы: `.wav`, `.ogg`
- Рекомендуется использовать короткие звуки (0.1-2 секунды)
- Громкость должна быть нормализована

## Примечание:

Если звуковые файлы отсутствуют, игра будет работать без звуков, но без ошибок.
